# Explainable AI for Credit Risk Prediction

This project predicts loan default risk using machine learning models on the German Credit (Statlog) dataset.

## Key Steps
- Performed data preprocessing, feature engineering, and exploratory data analysis (EDA)
- Trained models: Logistic Regression, Decision Tree, XGBoost
- Evaluated using ROC-AUC, Accuracy, and Confusion Matrix
- Applied SHAP for model interpretability

## Results
- ROC-AUC: 0.72 – 0.79
- Key factors: credit amount, loan duration, account status

## Dataset
German Credit (Statlog) dataset
